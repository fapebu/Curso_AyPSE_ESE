# Ejemplo blinking

Este ejemplo demuestra el funcionamiento de la arquitectura de capas utilizando la capa de Middleware de animación para ciclar entre secuencias de encendido de LEDs (parpadeo de todos los LEDs, secuencia simple de ida, y secuencia de ida y vuelta / ping-pong) en la placa ESP32-C6-DevKitC-1.
